## Folder structure

```
public
src
├── components
├── pages
│ └── api
```

## Coding Convention

- Folder naming: camelCase
- Components naming: PascalCase

## Prerequisites

- Node.js >= 10.13 (Recommended the latest version)
- (Yarn(Recommended))

## Install

```
yarn
# or
npm install
```

## Development

```
yarn dev
# or
npm run dev
```

## Build

```
yarn build
# or
npm run build
```

## Start

```
yarn start
# or
npm start
```
